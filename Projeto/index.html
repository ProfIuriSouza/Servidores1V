Métodos para Responsividade

Breakpoints

Utilização de infixos, são 6. Eles visam ajustar formatações para tamanhos de telas diferentes. O Bootstrap trabalha com o méto "Mobile First" ou seja, ele prioriza telas menores.

Containers

Permite criar regiões na página de largura fixa ou variável. Em todo caso, definem uma margem interna pré-definda.

A documentação Bootstrap indica para três classes: container (responsivo), container-fluid (tamanho fixo) e container-{breakpoint}.
